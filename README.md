# ros-pacman
SYSTEM REQUIREMENTS
	- Ubuntu 16.04 64 bits
	- ROS Kinetic
	- Catrkin workspace
	- Qt5 Development Libraries
	- OpenGL
		
COMPILATION
	- cd ~/catkin_ws/src
	- git clone https://github.com/carlosquinterop/ros-pacman.git
	- cd ..
	- source devel/setup.bash
	- catkin_make
	
EJECUCION
	- roscore
	- rosrun pacman pacman (in a new terminal)
